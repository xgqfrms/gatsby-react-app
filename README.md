# Gatsby React App

> gatsby-react-app

## install

```sh
$ npm i -g gatsby-cli
# or
$ sudo npm i -g gatsby-cli

```

## run

```sh
$ gatsby new gatsby-demo-app

$ cd gatsby-demo-app && gatsby develop
# auto open browser on `http://localhost:8000`

$ gatsby build
$ gatsby serve

$ gatsby -h
# or
$ gatsby --help

# gatsby COMMAND_NAME --help
$ gatsby new --help

```
